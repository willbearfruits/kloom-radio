## About
Kloom Lo Kadosh (Nothing Is Holy) is a radio project hosted by Yaniv Schonfeld. This repository serves as the static archive for past broadcasts, mixtapes, and experimental audio works.

It features:
- **Nothing Is Holy:** Mixtapes and live broadcasts (No Sleep For The Wicked, Creative Destruction).
- **Kol Hazuti (Visual Voice):** Classic interview series hosted by Noa Elran (2009-2010).
- **Kloom Originals:** Experimental audio and AI-generated content (The Bots' Religion).

## Technology
This site is a static archive generated by a custom Python build system.
- **Generator:** `generate.py` (Custom Python script).
- **Templating:** Jinja2.
- **Data:** `data/shows.json` (Single source of truth).
- **Integration:** Automatically fetches metadata (tags, images, descriptions) from the Mixcloud API during build.
- **Design:** "Glitch Brutalist" aesthetic (CSS).
